
# Smart Countdown with Arduino

This project was developed as part of the **Programming C++ for Engineers Using Arduino** course.

The Smart Countdown program demonstrates fundamental Arduino programming concepts by counting down from 5 to 1 while controlling the on-board LED. During each countdown step, the current count is displayed on the Serial Monitor, and the LED blinks a corresponding number of times. When the countdown is complete, a completion message is displayed.

## Features

* Countdown from 5 to 1 using a while loop
* LED blinking controlled through a custom function
* Serial Monitor output for real-time feedback
* Demonstrates basic embedded systems programming concepts

## Technologies Used

* Arduino IDE
* C++
* Arduino Uno

## Learning Objectives

This project demonstrates:

* Variables and data types
* Functions with parameters
* While loops
* Digital I/O operations
* Serial communication
* Hardware and software integration
